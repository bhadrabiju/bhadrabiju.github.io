Featured Projects

Fraudulent Job Offer Detection

Fraudulent Job Offer Detection is a machine learning–based system designed to identify fake job offer emails. The application analyzes the email subject and body using Natural Language Processing (NLP) techniques and combines them with indicators such as links, attachments, fee requests, and domain similarity to detect fraudulent patterns. The extracted features are processed using TF-IDF vectorization and classified using a Logistic Regression model to determine whether the job offer is fraudulent or genuine. The model is deployed through a Streamlit interface, allowing users to input email details and receive instant predictions.

🛠️ Tech Stack : Python | Pandas | NumPy | TF-IDF | Scikit-learn | Logistic Regression | Streamlit |

AI Karate Trainer for Hand Posture

🥋 AI Karate Trainer for Hand Posture AI Karate Trainer for Hand Posture is a computer vision–based application designed to validate karate hand postures in real time. The system analyzes hand landmarks to determine whether the hand is positioned correctly and displays the corresponding karate hand symbol only when the posture meets predefined criteria. This project focuses on posture accuracy rather than simple gesture recognition, helping users practice karate hand techniques independently without relying on a physical trainer.

🛠️ Tech Stack : Python | NumPy | OpenCV | MediaPipe | Hand Landmark
